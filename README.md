# Brevity

full-stack dapp template project using foundry and next.js

### Prerequisites

* yarn
* https://getfoundry.sh/

### Quickstart

```
yarn
yarn start
```

### Deployment

* Add the applicable secrets and vars to the github repo settings
* Create the project on vercel
